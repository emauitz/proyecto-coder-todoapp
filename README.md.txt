# Gestor de Tareas TODO-APP

Este proyecto implementa un Gestor de Tareas utilizando HTML, CSS y JavaScript. Permite a los usuarios registrar tareas, subtasks y marcarlas como completadas. Además, incluye una funcionalidad de barra de progreso para visualizar el avance de las tareas.

## Características

- **Registro y Gestión de Tareas:** Los usuarios pueden agregar nuevas tareas con descripciones detalladas.
- **Subtareas:** Cada tarea puede tener subtareas asociadas, facilitando la organización y la administración.
- **Marcado de Tareas Completadas:** Las tareas se pueden marcar como completadas, actualizando dinámicamente la barra de progreso.
- **Interfaz Responsiva:** Diseñado para ser compatible con dispositivos móviles y de escritorio.
- **Persistencia de Datos:** Utiliza `localStorage` para almacenar las tareas del usuario, asegurando que los datos permanezcan incluso después de cerrar el navegador.

## Uso

Para usar la aplicación:

1. Clona este repositorio en tu máquina local.
2. Abre `index.html` en tu navegador web.
3. Registra tus tareas y subtasks.
4. Marca las tareas completadas y observa la barra de progreso actualizar automáticamente.

## Tecnologías Utilizadas

- **HTML:** Estructura básica de la aplicación.
- **CSS:** Estilos y diseño responsivo.
- **JavaScript:** Lógica de la aplicación y manipulación del DOM.
- **Font Awesome:** Iconos utilizados en la interfaz de usuario.
- **SweetAlert2:** Para mensajes y alertas amigables.
- **Animate.css:** Animaciones para mejorar la experiencia del usuario.
- **Toastify:** Para notificaciones emergentes informativas.
- **Luxon:** Para calculo de dias y fecha actual.
